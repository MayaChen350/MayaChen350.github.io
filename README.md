## Build

To build the webste, if your system allows it, runs `make`.

Otherwise, do those steps:

1. Run `python` on `scripts/getJsonPoems.py` and `scripts/getNumerals.py`
2. Run `npm install`
3. Run `npm run build` if you want the static webpage or `npm run dev`
